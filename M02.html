    ガイドライン

    このシミュレーションでは、トポロジ内の各機器をコンソールから設定します。

    • 右の機器タブで設定対象の機器を切り替えます（採点対象の機器のみ表示されます）。
    • コマンドは省略形・?・Tab 補完に対応します。
    • 設定後は必ず copy running-config startup-config（=write）で保存してください。保存しないと0点です。
    • show running-config / show vlan brief / show ip interface brief / show interfaces status などの確認コマンドで現在の設定を確認できます（保存前でも可）。
    • 近隣探索は show cdp neighbors / show lldp neighbors、OSPF隣接は show ip ospf neighbor で簡易確認できます。
    • 本演習は到達性（接続テスト）を模擬しません。
